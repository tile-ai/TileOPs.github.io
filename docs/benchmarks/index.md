
# Benchmark Results

Performance measurements for TileOPs operators across hardware and configurations.

!!! note
    Benchmark results are continuously updated from nightly CI runs. Numbers reflect the latest validated results on our benchmark hardware.


## Regression Detection

- **Threshold**: >10% latency regression vs 7-day rolling median triggers alert
- **Correctness**: Headroom ratio >0.8 triggers drift warning
- **Test time**: >20% regression in benchmark duration flagged
